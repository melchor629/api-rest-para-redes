Bienvenido a la página principal

Esta es la página inicial (index.html) de la aplicación TodoList, que sirve para demostrar que en un servidor (Servidor Web) que se ofrece una API RESTFul también se sirven conjuntamente páginas web.

Descargar programa de muestra
Contiene la biblioteca y la GUI. Para ejecutar 

java -cp todoListApp.jar gui.MainWindow

Descargar código fuente del programa de muestra
Contiene todo el código fuente, la documentación JavaDoc y el proyecto Gradle para compilar. No usar interfaz gráfica si no estás usando IntelliJ Idea, la GUI requiere de un componente que tiene dicho IDE.
